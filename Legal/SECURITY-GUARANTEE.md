## The Architecture Guarantee

QuickBooks Time-Warp® was **architecturally designed** to make data leakage impossible:

- 🔒 **No network calls** — The migration engine does not make any internet connections
- 🔒 **No telemetry** — We do not phone home, ping servers, or send analytics
- 🔒 **No cloud dependencies** — The software works entirely offline
- 🔒 **No remote APIs** — All processing uses the local QuickBooks SDK (QBXML)
- 🔒 **No temp uploads** — Working files stay in a local directory you control
- 🔒 **Open audit trail** — Every operation is logged locally for your review

This isn't a policy decision that could change — it's a **technical architecture** that makes data collection physically impossible.
